# Workflow: Extract

Use this after transcription when the user wants summary, notes, or content reuse.

## Modes

- Quick summary: 5-8 core points, concise.
- Structured notes: sections, bullets, examples, key quotes.
- Content draft: turn the transcript into a publishable article or post.

## Input

Use the transcript text from the previous command, or read the transcript file path returned by the CLI.

## Output

Preserve useful original wording. Do not invent details not present in the transcript.
